# PyTorch Building Blocks

Implementations of core PyTorch operations and neural network components from first principles, using only `torch.Tensor` operations (no `nn.Module` shortcuts).

## math_you_need_for_dl_code/

| File | Description |
|------|-------------|
| `autograd.py` | Automatic differentiation with `torch.autograd` |
| `cosine_similarily.py` | Cosine similarity between vectors |
| `crossentropyloss.py` | Cross-entropy loss computation |
| `dropout.py` | Dropout during training |
| `embedding.py` | Token embedding lookup |
| `flatten.py` | Tensor flattening |
| `gradient_descent.py` | Gradient descent optimizer |
| `linear.py` | Linear (fully connected) layer |
| `multiheadattention.py` | Multi-head attention mechanism |
| `parameter.py` | PyTorch parameter handling |
| `positional_encoding.py` | Sinusoidal positional encoding |
| `relu.py` | ReLU activation |
| `resnet.py` | Residual network block |
| `scaled_dot_product.py` | Scaled dot-product attention |
| `single_neuron.py` | Single neuron forward pass |
| `training_loop.py` | Training loop pattern |
| `transformer_decoder.py` | Transformer decoder block |
| `transformer_encoder.py` | Transformer encoder block |
| `normalisation/batchnorm1d.py` | 1D batch normalization |
| `normalisation/layernorm.py` | Layer normalization |
